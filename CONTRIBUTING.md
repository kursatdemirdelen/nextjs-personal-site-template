# Contributing

Thanks for your interest in contributing!

## How to Contribute

### Reporting Bugs

Found a bug? [Open an issue](https://github.com/kursatdemirdelen/nextjs-personal-site-template/issues) with:
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

### Feature Requests

Have an idea? Open an issue describing:
- What problem it solves
- Your suggested approach

### Code Contributions

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Commit: `git commit -m "Add feature description"`
5. Push: `git push origin feature/your-feature`
6. Open a Pull Request

## Development Setup

```bash
git clone https://github.com/YOUR-USERNAME/personal-site.git
cd personal-site
npm install
npm run dev
```

## Commit Guidelines

Use descriptive commit messages:
- `feat: add dark mode toggle`
- `fix: correct blog post sorting`
- `docs: update README`
- `refactor: simplify header component`

## Testing

Before submitting:
```bash
npm run build  # Ensure build works
npm run lint   # Check for linting errors
```

Thanks for contributing!
